Provides an alternative link to the blog archive site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/mhm-404-link` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. You can now add the block to the editor.
